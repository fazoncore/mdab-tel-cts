# Security Policy

## Reporting a Vulnerability
If you believe you have found a security vulnerability in this repository (e.g.,
issues in the verifier, signature checks, parsing, or fixtures), please report
it privately.

- Preferred: open a GitHub Security Advisory (private disclosure)
- Alternative: email the maintainers listed in `MANIFEST.md`

Please include:
- steps to reproduce
- impacted files
- expected vs actual behavior

## Scope
This repository includes reference implementations for verification and testing.
It is not a production security boundary by itself.
